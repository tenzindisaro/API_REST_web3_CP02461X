API REST com Node.js (Microsserviços)
📌 Descrição

Projeto de API REST utilizando Node.js, Express, Sequelize e MySQL com arquitetura de microsserviços.
O sistema possui três serviços independentes que se comunicam via HTTP.

🧩 Serviços

cliente-service → gerenciamento de clientes

produto-service → gerenciamento de produtos

pedido-service → gerenciamento de pedidos (valida clientes e produtos)

⚙️ Tecnologias

Node.js/Express/Sequelize/MySQL/dotenv

📡 Endpoints principais

Clientes → http://localhost:3001/clientes
Produtos → http://localhost:3003/produtos
Pedidos → http://localhost:3002/pedidos
